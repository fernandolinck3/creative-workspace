# three-utils

Cenas 3D e presets de GSAP reutilizáveis entre os 3 apps:
- scenes/ — partículas, blob, globo, geometria flutuante
- gsap-presets/ — scroll-reveal, parallax, text-reveal
- hooks/ — detecção de device, prefers-reduced-motion

Regra: qualquer padrão usado em mais de 1 app deve morar aqui, não
duplicado dentro de cada app.
