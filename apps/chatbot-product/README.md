# chatbot-product

Landing de venda do produto de automação de chatbot (WhatsApp) para
negócios locais. Reaproveita o agente whatsapp-sdr já desenhado no
ecossistema de agentes autônomos.

## Status
- [ ] Pricing/planos definidos
- [ ] Demo interativa (chat de exemplo na página)
- [ ] Primeiros cases piloto

## Stack
React + Vite + Tailwind + Three.js/R3F + GSAP (via packages/ui-kit e packages/three-utils)
